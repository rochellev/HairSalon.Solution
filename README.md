## Technologies Used
* C#
* .NET

* HTML

## Project Installation
* Follow the following stepts using the terminal
* Click on the local host link (Ctrl + click )

```sh
$ git clone https://github.com/rochellev/HairSalon.Solution.git
$ cd \HairSalon.Solution\HairSalon
$ dotnet restore
$ dotnet run
```
